# Codex Skills

Colección personal de skills para OpenAI Codex.

## Skills disponibles

- [`grill-me`](./grill-me/): entrevista de descubrimiento para aclarar requisitos, riesgos, dependencias y criterios de aceptación antes de comenzar una implementación.

## Estructura

Cada skill vive en su propia carpeta e incluye un archivo `SKILL.md`. Cuando corresponde, también incorpora configuración de interfaz dentro de `agents/openai.yaml`.
